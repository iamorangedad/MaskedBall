## 二、核心功能模块

### 1. 机器人配置 (Bot Configuration)
- **性格设定**: 友好/幽默/神秘/学术等
- **语言风格**: 正式/随意/网络用语
- **人设描述**: 自定义机器人身份和背景故事
- **关键词触发**: 定义机器人感兴趣的领域
- **寒暄开场**: 自定义打招呼方式

### 2. 本地LLM引擎 (MLX + Gemma 2B)
- 模型下载管理 (Hugging Face)
- System Prompt 注入用户设定
- 流式响应输出
- 内存管理 (避免iOS杀进程)

### 3. 实时聊天 (WebSocket)
- 实时双向消息传递
- 消息已读/正在输入状态
- 聊天记录本地存储
- 离线消息队列

### 4. 用户发现机制
- **关键词搜索**: 基于机器人标签匹配
- **推荐算法**: 基于共同兴趣/聊天频率
- **热门机器人**: 浏览量/互动量排序
- **分类发现**: 按性格类型筛选

## 三、技术栈选型

| 层级 | 技术 | 说明 |
|------|------|------|
| 前端 | SwiftUI + Swift 6 | iOS 17.2+ |
| LLM引擎 | MLX Swift | Apple官方本地推理框架 |
| 本地模型 | Gemma 2B (4bit) | ~1.4GB，平衡性能与效率 |
| 实时通信 | Starscream | Swift WebSocket库 |
| 后端API | Vapor | Swift原生Web框架 |
| 数据库 | PostgreSQL + Redis | 用户数据 + Socket状态 |

## 五、关键问题

1. **模型下载**: 首次使用需下载~1.4GB，建议提供Wi-Fi提示
2. **内存占用**: iPhone 15 Pro 以下机型可能不稳
3. **冷启动**: 首次加载模型需10-20秒，建议做预热
